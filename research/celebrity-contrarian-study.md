# 名人效應逆向指標策略研究

> 日期：2026-04-10
> 狀態：初步研究（回測前的可行性評估）
> 目標：觀察市場名人公開發言是否有可量化的散戶效應，能否作為策略指標

---

## 1. Jim Cramer（美國）

### 背景
- CNBC《Mad Money》主持人，1955年生
- 前對沖基金經理（Cramer & Co.），聲稱14年年化24%
- 全美最具影響力的散戶投資意見領袖之一
- 每日推薦多檔股票，數據量最大，追蹤最完整

### 已知數據

**Wharton/Northwestern 學術研究（Bolster & Triana, 2009）**
- 追蹤期間：2005-2008
- 買入推薦：短期有正面效應（推薦後1-2天），但**長期跑輸大盤**
- 結論：Cramer 的推薦製造短期流動性衝擊，但沒有長期 alpha

**Bankrate 統計（2023-2024）**
- Cramer 2023年上半年推薦的27檔股票：
  - 正面推薦（買入）：平均表現與 S&P 500 基本持平
  - 但 **個股波動性遠大於指數**，風險不成比例
- 結論：「跟著 Cramer 買不一定賺，但風險肯定比指數高」

**Inverse Cramer ETF 實驗（SJIM）**
- Tuttle Capital 於 2023年3月推出 SJIM（做空 Cramer 推薦的 ETF）
- **結果慘烈**：SJIM 從上市至 2026-04-09 跌了 ~39.5%
- 同期 SPY 漲了 ~46.5%
- **Inverse Cramer 做為持續策略：失敗。**
- SJIM 最終在 2024-2025 間清盤

**Quiver Quantitative 追蹤器**
- 追蹤 Cramer 在 Mad Money 上所有的 buy/sell calls
- 資料庫完整，可量化分析
- 整體結論：Cramer 推薦的「短期反轉效應」最明顯
  - 推薦當天/次日：異常正報酬（散戶湧入）
  - 1週後：效應消退
  - 1-3個月後：回歸均值，部分跑輸大盤

### Cramer 效應時間模式
```
推薦日    +1天     +1週     +1月     +3月     +6月
  │       │        │        │        │        │
  ▼       ▼        ▼        ▼        ▼        ▼
 散戶     短期    效應     回歸     部分     無顯著
 湧入     +1~3%   消退     均值     跑輸     差異
```

### 產業特徵
- **科技股**：Cramer 效應最強（散戶參與度高）
- **大型藍籌**：效應較弱（機構主導）
- **迷因股/SPAC**：效應最不可預測（已有極端情緒）

### 策略可行性：⭐⭐⭐ (3/5)
- **短期做空 Cramer 新推薦**：有理論基礎但 SJIM 已證明持續做空不行
- **買 Cramer 推薦+1天後賣**：需要極低交易成本
- **最佳用法**：Cramer 大量推薦某產業 = 該產業散戶過熱信號

## 2. 吃土鋁神巴逆逆（台灣）

### 背景
- 台灣網路股市 KOL
- 公開搜尋資料極少，無法找到系統性的推薦記錄
- 可能主要活躍於 PTT Stock 板、Dcard、或特定社群

### 數據現況
- ❌ 無公開統計數據
- ❌ 無第三方追蹤器
- ❌ 搜尋引擎幾乎無結果
- 需要手動從社群媒體爬取歷史發言才能回測

### 策略可行性：⭐ (1/5)
- 數據取得困難，無法做有意義的統計回測
- 如果未來要追蹤，需要建立社群媒體爬蟲定期收集發言

## 3. 謝金河（台灣）

### 背景
- 《財訊》前總編輯，現為財信傳媒集團董事長
- 台灣最具影響力的財經評論家之一
- 主要發言管道：Facebook「老謝看世界」、電視節目、財訊雜誌
- 風格：宏觀趨勢導向，較少做個股「買入/賣出」推薦
- 更傾向產業趨勢、政策解讀、大盤方向

### 數據現況
- ❌ 無系統性的第三方統計追蹤
- 謝金河的發言通常是「看好某產業」而非「買入某股」
- Facebook 貼文是主要數據來源，但需要手動爬取
- 有些零星的 PTT/Dcard 討論，但都是個別案例而非統計

### 已知模式（定性觀察）
- 謝金河看好 AI/半導體供應鏈：2023-2024年整體方向正確
- 謝金河看好電動車：2022年看好，但電動車股 2023 表現分化
- 風格偏保守，發言是「我認為趨勢」而非「我建議買入」
- 散戶追隨度高，但因為他的建議通常是產業級別，效應難以量化到個股

### 策略可行性：⭐⭐ (2/5)
- 產業方向性參考價值 > 個股推薦價值
- 如果他在 Facebook 密集提到某產業 = 散戶已經在追了
- 需要 NLP 分析他的 Facebook 貼文才能量化

## 4. 谷月涵 Peter Kurz（台灣）

### 背景
- 美國人，花旗環球前台灣區研究部主管
- 在台灣財經圈被稱為「乖乖牌外資分析師」
- 長期參與台灣財經媒體討論
- 風格：外資觀點解讀台股，宏觀導向

### 數據現況
- ❌ 無系統性的第三方統計追蹤
- 主要出現在財經節目和新聞採訪
- 發言多為大盤方向和外資動向，非個股推薦
- 有「谷月涵看多/看空」的新聞報導可追溯

### 已知模式（定性觀察）
- 通常反映外資對台灣市場的看法
- 對大盤方向的判斷能力 > 個股選擇
- 發言頻率較低（相比 Cramer 或謝金河）
- 風格穩健，較少做極端看多/看空表態

### 策略可行性：⭐⭐ (2/5)
- 作為外資情緒的代理指標有一定價值
- 數據量太少，難以做統計顯著的回測
- 需要手動建立發言資料庫

## 6. 核心發現

### Jim Cramer 是唯一可量化回測的對象

原因：
1. Quiver Quantitative 有完整的歷史推薦記錄（ticker + date + direction）
2. SJIM/LJIM ETF 提供了市場級別的驗證
3. 有學術研究佐證
4. 數據量夠大（每年數百個推薦）

### 「持續做空名人推薦」策略已被證偽

SJIM 的失敗告訴我們：
- 名人推薦不是穩定的反指標
- Cramer 的推薦平均不差（接近市場報酬）
- 做空的成本（借券費、逆風期）吃掉了任何可能的 alpha

### 真正有價值的信號：過熱/恐慌檢測

名人效應的最佳用法不是「反著做」，而是：
1. **過熱信號**：多位名人同時看多某產業 → 散戶已過度集中 → 謹慎
2. **恐慌信號**：名人集體轉空 → 散戶恐慌拋售 → 可能是買點（與 S4 策略一致）
3. **共識破裂**：本來一致看多突然分歧 → 趨勢可能轉折

## 7. 可行的實驗方案

### 方案 A：Cramer 短期動量衰減（可立即回測）

```
觸發：Cramer 在 Mad Money 推薦買入某股
進場：推薦次日 close（散戶已推高）
出場：+5 交易日後 close
方向：做空（賭散戶動量衰減）
停損：+3% 以上
```

**數據來源**：Quiver Quantitative API 或 CSV
**回測期間**：2020-2025（含多種市場環境）
**所需工具**：yfinance + pandas

### 方案 B：台灣 KOL 情緒聚合指標（需先建資料庫）

```
每週五統計：
- 謝金河 FB 本週提到的前5大產業
- 財經節目出現頻率最高的個股
- PTT Stock 板熱門股
→ 產生「散戶熱度指數」
→ 熱度 >80% → 該產業過熱警告
→ 熱度從 >80% 急降至 <30% → 恐慌買點信號
```

**前提**：需要建立 Facebook/PTT 爬蟲（工程量大）
**時程**：2-3 週

### 建議：先跑方案 A

Cramer 數據現成，可以在 30 分鐘內完成初步回測。如果效應顯著，再投資時間建台灣的數據管道。

## 8. 對 PAI Fund 的潛在貢獻

如果研究結果正向，可以作為 **S6: Celebrity Contrarian Signal**：

```
定義：名人共識指標（Celebrity Consensus Index, CCI）
計算：追蹤 N 位名人對特定產業/個股的正面/負面發言密度
應用：
  CCI > 0.8（所有人看多）→ 觸發產業過熱警告，減持信號
  CCI < 0.2（所有人看空）→ 觸發恐慌信號，與 S4 聯動
  CCI 急速反轉（0.8 → 0.3）→ 趨勢轉折信號
```

這不是獨立的交易策略，而是**風險疊加指標**，增強現有 S3（市場週期）和 S4（恐慌反彈）的信號質量。
