## 2. 死不炸去 / DieWithoutBang（台灣）

### 背景
- Facebook 粉專「死不炸去 DieWithoutBang」
- 台灣股市 KOL，Facebook 追蹤者數不詳
- Facebook 需要登入才能看到完整貼文內容

### 數據現況
- ❌ Facebook 粉專需要登入才能瀏覽
- ❌ 無公開第三方統計追蹤
- ❌ 搜尋引擎無法抓取 Facebook 粉專內容
- 需要 Facebook Graph API 或手動爬取才能建立資料庫

### 如何建資料庫
1. 方案 A：人工翻閱粉專貼文，記錄每則提到個股的貼文（日期 + ticker + 正面/負面）
2. 方案 B：Facebook Graph API（需要粉專管理者授權）
3. 方案 C：第三方 Facebook 爬蟲工具（法律灰色地帶）

### 策略可行性：⭐ (1/5) — 需要先建資料庫

## 5. 其他已知的名人指標

### 🏆 Nancy Pelosi / 美國國會議員交易（最強正向指標）

**這是目前學術上證據最強的「名人」交易指標。**

- **Unusual Whales 2024 年度報告**：
  - 2024 年國會議員交易整體 **跑贏 S&P 500 約 18%**
  - Nancy Pelosi 2024 年報酬率 **+54%**（S&P 500 同期 +23%）
  - 眾議院 > 參議院（眾議員交易表現更好）
  - 科技股是國會議員最愛的產業

- **為什麼有效**：
  - 國會議員有非公開資訊（聽證會、法案內容、監管方向）
  - 信息不對稱是合法的（STOCK Act 執行力度弱）
  - 他們的交易必須在 45 天內公開揭露
  - 揭露後仍有 alpha（市場反應不足）

- **追蹤工具**：
  - Unusual Whales（unusualwhales.com）— 最完整
  - Quiver Quantitative — 免費 API
  - Capitol Trades（capitoltrades.com）
  - STOCK Act 揭露文件（公開）

- **策略可行性：⭐⭐⭐⭐⭐ (5/5)**
  - 數據公開、結構化、可自動追蹤
  - 有學術研究支撐
  - 有現成 ETF（NANC, KRUZ）
  - 歷史表現顯著

### 📊 Michael Burry（反向情緒指標）

- 以做空成名（次貸危機 The Big Short）
- 13F 揭露持倉，每季更新
- 近年做空 NVDA、PLTR 等 AI 股 → 目前看起來是錯的
- 長期做空美股的人在牛市都是反指標
- **策略價值**：Burry 做空某股 → 不要跟做空，但列入觀察
- **策略可行性：⭐⭐⭐ (3/5)** — 數據可得但頻率低（季度）

### 📊 Cathie Wood / ARK Invest（散戶風向標）

- ARK 每日公開交易紀錄（免費下載 CSV）
- 2020-2021 是正向指標（大牛市科技股）
- 2022-2023 是反向指標（高估值崩潰）
- ARKK ETF 從 159 跌到 30（-81%），然後反彈到 50-60
- **策略價值**：ARK 大量買入 = 散戶追捧的東西，用作情緒指標
- **策略可行性：⭐⭐⭐⭐ (4/5)** — 每日數據、完全公開、CSV 格式

### 📊 WSB / Reddit WallStreetBets（散戶集體情緒）

- 有追蹤工具：Quiver Quantitative WSB tracker
- GME、AMC 等迷因股的源頭
- 學術研究顯示 WSB 提及頻率與短期股價波動正相關
- **策略價值**：WSB 某股提及量暴增 = 短期動量但中期反轉
- **策略可行性：⭐⭐⭐⭐ (4/5)** — API 可得、歷史數據完整

### 📊 雜誌封面指標（經典反向指標）

- The Economist、Bloomberg Businessweek、Time 封面
- 學術研究（2007, Wired）：雜誌封面看多某公司後，該公司平均跑輸大盤
- 原理：封面 = 故事已經被充分定價
- **策略價值**：定性指標，封面看多 = 謹慎
- **策略可行性：⭐⭐ (2/5)** — 頻率太低，難以系統化

## 8. 名人共識指標（Celebrity Consensus Index, CCI）設計

不做獨立策略，作為現有策略的信號增強器：

```
CCI 計算 = weighted_sum(
    congress_buy_count × 3,    # 權重最高（有信息優勢）
    ark_position_change × 1,   # 散戶風向
    cramer_sell_flag × 2,      # 賣出信號較準
    wsb_mention_spike × 1,     # 短期過熱
)

CCI > 0.8 → 散戶過熱警告，搭配 S3 使用
CCI < 0.2 → 恐慌信號，搭配 S4 使用
Congress + Cramer SELL 同方向 → 高度重視
```
