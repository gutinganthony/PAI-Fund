# S5: 趨勢跟隨策略（加密專用）

**來源**：Minervini Stage Analysis（改良） + 多時間框架均線系統
**市場**：加密貨幣（OKX）
**時間框架**：4H + 日線
**風格**：趨勢跟隨 — 跟著趨勢走，回調時加倉

## 策略邏輯

使用多條均線判斷趨勢方向和強度，在回調至支撐均線時進場，趨勢破壞時出場。專為加密貨幣 24/7 市場設計。

## 均線系統

```
EMA 21（快線）— 短期趨勢
EMA 55（中線）— 中期趨勢
EMA 200（慢線）— 長期趨勢

多頭排列：EMA21 > EMA55 > EMA200（全部上升）
空頭排列：EMA21 < EMA55 < EMA200（全部下降）
```

## 進場規則

### 多頭進場（做多）
1. 日線：EMA21 > EMA55 > EMA200（多頭排列確認）
2. 日線：EMA200 連續 10 日上升
3. 4H線：價格回調至 EMA21 或 EMA55 附近（±1.5%）
4. 4H線：出現看漲 K 線形態（錘子、吞噬、早晨之星）
5. 成交量：回調期間縮量，反彈時放量

### 進場觸發
6. 4H 線收盤價站上 EMA21 且高於前一根 4H 高點

## 出場規則

| 條件 | 動作 | 時間框架 |
|------|------|---------|
| 買入價 -10% | 全部出場（硬停損） | 任何 |
| 日線 EMA21 下穿 EMA55 | 賣出 1/2 | 日線 |
| 日線 EMA55 下穿 EMA200 | 全部出場 | 日線 |
| 4H RSI > 80 + 量價背離 | 賣出 1/3 | 4H |
| 追蹤停損：最高價 -8% | 全部出場 | 日線 |

## 追蹤停損機制

```
初始停損 = 買入價 × 0.90（-10%）

盈利 > 10% → 停損上移至買入價（保本停損）
盈利 > 20% → 停損上移至買入價 × 1.10（鎖住 10%）
盈利 > 30% → 停損 = 最高價 × 0.92（追蹤 -8%）

停損只能上移，永遠不能下移
```

## 倉位管理

- 單幣種最大倉位：策略資金 × 40%
- 單筆風險：策略資金 × 2%
- 最大同時持有：4 個幣種
- 加倉規則：首次建 50% 倉位，回調確認後加至 100%

## 不交易條件

- 日線均線糾結（EMA21/55/200 差距 < 2%）→ 盤整，不操作
- BTC 日線 ATR(14) < 近 3 月平均的 50% → 低波動期，等待
- 重大事件前 24 小時（FOMC、CPI 公布）→ 暫停進場

## 監控頻率

- 每 4 小時：檢查 4H 線信號
- 每日：更新日線均線狀態、計算 ATR
- 即時：停損單常駐（掛單不撤）

## 監控指標
- 勝率目標：> 40%
- 盈虧比目標：> 3:1
- 月度最大回撤：-10%
- 平均持有期：5-20 天
