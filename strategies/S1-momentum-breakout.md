# S1: 動能突破策略

**來源**：Mark Minervini SEPA + VCP
**市場**：加密貨幣（OKX）+ 美股（Alpaca）
**時間框架**：日線
**風格**：趨勢跟隨 — 買入突破、嚴格停損

## 策略邏輯

在確認的上升趨勢中（Stage 2），等待價格波動收斂（VCP），在放量突破時進場。

## 進場規則

### 趨勢確認（SEPA 模板，全部滿足）
1. 價格 > 50MA > 150MA > 200MA
2. 200MA 連續 22 個交易日走升
3. 價格距 52 週低點 > +30%
4. 價格距 52 週高點 < -25%

### VCP 確認
5. 至少 2 次價格回調，每次幅度遞減
6. 最後一次回調幅度 < 前一次的 60%
7. 收斂期間成交量逐步萎縮

### 進場觸發
8. 價格突破 VCP 最高點
9. 突破日成交量 > 50 日均量 × 1.5

## 出場規則

| 條件 | 動作 | 備註 |
|------|------|------|
| 買入價 -7%（美股）/ -12%（加密） | 全部出場 | 硬停損，不可商量 |
| 盈利 +20% | 賣出 1/3 | 鎖住部分利潤 |
| 跌破 50MA | 賣出 1/2 | 趨勢減弱 |
| 跌破 150MA | 全部出場 | 趨勢結束 |
| 一日暴漲 >15% + 巨量 | 賣出 1/2 | Climax top 信號 |

## 倉位管理

```
單筆風險 = 策略資金 × 2%
倉位大小 = 單筆風險 ÷ (買入價 - 停損價)

範例（美股，策略資金 $20,000）：
  買入價 = $100
  停損價 = $93（-7%）
  單筆風險 = $20,000 × 2% = $400
  倉位大小 = $400 ÷ $7 = 57 股
  倉位金額 = $5,700（佔策略資金 28.5%）

範例（加密，策略資金 20,000 USDT）：
  買入價 = $60,000 (BTC)
  停損價 = $52,800（-12%）
  單筆風險 = 20,000 × 2% = 400 USDT
  倉位大小 = 400 ÷ 7,200 = 0.055 BTC
```

## 加密貨幣特殊調整
- 停損放寬至 10-12%（波動更大）
- 使用 4H 線輔助判斷（日線為主）
- 標的限制：BTC, ETH, SOL, + 市值前 30 的幣種
- 不做合約/槓桿（模擬階段）

## 監控指標
- 勝率目標：> 45%
- 盈虧比目標：> 2.5:1
- 最大連續虧損：5 筆（觸發暫停）
- 月度最大回撤：-8%（觸發減半倉位）
