# S3: 市場週期策略

**來源**：Kostolany 雞蛋理論 + Howard Marks 週期溫度計
**市場**：加密貨幣（OKX）+ 美股（Alpaca）
**時間框架**：週線/月線
**風格**：逆向週期 — 在極端時刻反向操作

## 策略邏輯

綜合多個情緒和週期指標計算「市場溫度」，在過冷時加倉、過熱時減倉。不做個股選擇，只做指數/大盤級別的倉位管理。

## 市場溫度計算

### 美股溫度

```
美股溫度 = (
    Fear_Greed_Index × 0.30 +
    (100 - VIX_Normalized) × 0.25 +
    Put_Call_Score × 0.20 +
    Margin_Debt_Score × 0.15 +
    Yield_Curve_Score × 0.10
)

VIX 正規化：
  VIX < 12 → 95（極度自滿）
  VIX 12-20 → 60-80
  VIX 20-30 → 30-50
  VIX 30-40 → 15-25
  VIX > 40 → 5（極度恐慌）
```

### 加密溫度

```
加密溫度 = (
    Crypto_Fear_Greed_Index × 0.35 +
    BTC_RSI_Monthly × 0.25 +
    Funding_Rate_Score × 0.20 +
    Exchange_Reserve_Score × 0.10 +
    Stablecoin_Flow_Score × 0.10
)

Funding Rate 評分：
  < -0.01% → 10（極度恐慌/空頭過度）
  -0.01% ~ 0.01% → 50（中性）
  0.01% ~ 0.05% → 70（偏樂觀）
  > 0.05% → 90（極度貪婪/多頭過度）
```

## 倉位規則

| 溫度區間 | 市場狀態 | 目標倉位 | 操作 |
|---------|---------|---------|------|
| 0-20 | 極度恐慌 | 80-90% | 積極買入 |
| 20-40 | 偏冷 | 60-70% | 逐步加倉 |
| 40-60 | 中性 | 40-50% | 維持 |
| 60-80 | 偏熱 | 20-30% | 逐步減倉 |
| 80-100 | 極度貪婪 | 0-10% | 防守模式 |

## 交易標的

### 美股（Alpaca）
- SPY（S&P 500 ETF）— 主力
- QQQ（NASDAQ 100 ETF）— 科技偏重
- 不做個股，只做指數

### 加密貨幣（OKX）
- BTC — 佔加密倉位 60%
- ETH — 佔加密倉位 30%
- SOL — 佔加密倉位 10%

## 調倉規則

- 溫度變化 > 10 點且跨區間 → 調整倉位
- 每次調倉不超過總倉位的 20%（避免一次性大動作）
- 調倉間隔至少 3 天（避免頻繁操作）
- 溫度在 40-60 區間 → 不操作

## 監控頻率

- 每日：計算溫度，記錄到日誌
- 每週：評估是否需要調倉
- 溫度劇變（單日變化 > 15 點）→ 立即評估

## 監控指標
- 年化回報目標：> 10%
- 最大回撤目標：< -15%
- 溫度預測準確率（事後驗證）
- 調倉次數/年：目標 8-15 次（不過度頻繁）
