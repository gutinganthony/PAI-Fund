# 策略自我辯證框架

> 「The biggest investing errors come not from factors that are informational or analytical, but from those that are psychological.」— Howard Marks

## 原則

每一個交易決策都必須經過「紅隊」挑戰。不是找理由買，而是**找理由不買**，然後看這些理由是否站得住腳。

## 辯證模板

### 對每筆交易問三個問題：

1. **如果我錯了，最壞的結果是什麼？我能承受嗎？**
2. **有什麼我可能忽略的因素會讓這筆交易失敗？**
3. **我是在跟隨系統信號，還是在合理化自己想買的衝動？**

### 對每個策略問三個問題：

1. **這個策略在什麼環境下會失效？現在是否接近那個環境？**
2. **如果所有人都在用類似策略，還會有效嗎？（擁擠度風險）**
3. **過去 3 個月的表現是否源自真正的邊際優勢，還是運氣？**

## AVI 3.0 對系統的影響

### 當前 AVI 讀數：5.67（中等偏高）

| 維度 | 得分率 | 對策略的影響 |
|------|--------|-------------|
| 估值 96% 🔴 | 美股整體偏貴 | 不追高、只在大折扣時進場 |
| 獲利 20% 🟢 | 企業基本面健康 | 支撐估值不至於崩盤 |
| 宏觀 40% 🟡 | VIX 偏高但可控 | 波動可能加大 |
| 信用 11% 🟢 | 信用極度健康 | 沒有 2007 式系統危機跡象 |
| 動量 36% 🟢 | 已回調，風險部分釋放 | 有利於選擇性進場 |

**AVI 的核心訊息：估值高但不是危機。信用健康是最大的安全網。**

這意味著：
1. 不要大規模建倉（估值 96%）
2. 但可以選擇性進場大折扣的優質標的（信用 11% = 沒有系統性風險）
3. 關注 AVI 趨勢 — 如果信用維度開始上升，那就真的要小心了

## 新策略實驗構想

### 實驗 1：AVI 驅動的倉位管理（整合進 S3）

用 AVI 替代或輔助 Howard Marks 的溫度計：

```
如果 AVI > 7.0 → S3 倉位上限 20%
如果 AVI 6.0-7.0 → S3 倉位上限 40%
如果 AVI 5.0-6.0 → S3 倉位上限 60%（當前）
如果 AVI 4.0-5.0 → S3 倉位上限 80%
如果 AVI < 4.0 → S3 倉位上限 100%
```

優勢：AVI 用 12 個指標 × 歷史百分位，比我們自己的溫度計更嚴謹。

### 實驗 2：維度分歧策略

當 AVI 的不同維度發出矛盾信號時，反而可能是機會：
- 估值高 + 信用低 = 「貴但不危險」→ 選擇性買入折扣標的
- 估值低 + 信用高 = 「便宜但有系統風險」→ 要更謹慎
- 全部高 = 離場
- 全部低 = 全力進場

### 實驗 3：AI 主題 vs 傳統價值輪動

追蹤 AI 供應鏈股（NVDA, CEG, VST）vs 傳統價值股（BRK-B, COST, V）的相對表現：
- 如果 AI 股持續跑贏 → Leopold 論點被市場驗證
- 如果傳統價值股跑贏 → 市場在去 AI 泡沫化
- 根據輪動方向調整配置
