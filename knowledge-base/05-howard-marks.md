# Howard Marks 霍華德·馬克斯

> 「你無法預測，但你可以準備。」

## 核心理念

**一句話**：卓越的投資來自於理解風險、逆向思考、以及在市場週期的極端時刻採取行動。

### 五大核心原則

1. **第二層思考（Second-Level Thinking）** — 不問「會發生什麼」，而問「大家認為會發生什麼，以及如果他們錯了會怎樣」
2. **理解週期** — 萬物皆有週期，沒有什麼趨勢是永恆的。關鍵是判斷我們處於週期的哪個位置
3. **風險不等於波動** — 真正的風險是永久性資本損失，不是價格波動
4. **逆向投資** — 最好的機會出現在大家都不想買的時候
5. **知道自己不知道什麼** — 承認不確定性，據此調整部位大小

## 第二層思考框架

| 層級 | 思考方式 | 範例 |
|------|---------|------|
| 第一層 | 「這家公司很好，買入」 | 看到好公司就買 |
| 第二層 | 「這家公司好，但所有人都知道，已經反映在價格裡了」 | 好公司不一定是好投資 |
| 第一層 | 「經濟要衰退了，賣出」 | 聽到壞消息就恐慌 |
| 第二層 | 「所有人都在拋售，恐慌已超過實際情況，買入」 | 別人的恐慌是你的機會 |

### 量化應用
- 當共識極端一致時（>80% 看多/看空），考慮反向
- 用 CNN Fear & Greed Index / Crypto Fear & Greed Index 量化市場情緒
- 用 Put/Call Ratio 量化選擇權市場情緒

## 市場週期溫度計

Marks 的核心工具：判斷我們處於週期的哪個位置

```
溫度: 0 ────────── 50 ────────── 100
      過冷(買入)    中性(持有)    過熱(減倉)

量化指標對應：
- Fear & Greed Index:   0-25=過冷  25-75=中性  75-100=過熱
- VIX:                  >35=過冷   15-35=中性   <15=過熱
- Crypto F&G:           0-20=過冷  20-80=中性   80-100=過熱
- Put/Call Ratio:       >1.2=過冷  0.7-1.2=中性 <0.7=過熱
- Margin Debt Growth:   下降=過冷  穩定=中性    急增=過熱
```

## 可量化規則

### 週期溫度計算（綜合評分）

```python
temperature = (
    fear_greed_index * 0.30 +
    (100 - normalize_vix(vix)) * 0.25 +
    crypto_fear_greed * 0.20 +
    (100 - normalize_put_call(put_call_ratio)) * 0.15 +
    margin_debt_score * 0.10
)
```

### 行動規則
- `溫度 < 20`（極度恐慌）→ 積極買入，倉位拉到 80-90%
- `溫度 20-40`（偏冷）→ 逐步加倉至 60-70%
- `溫度 40-60`（中性）→ 維持 50% 倉位，正常操作
- `溫度 60-80`（偏熱）→ 逐步減倉至 30-40%
- `溫度 > 80`（極度貪婪）→ 防守模式，倉位降至 10-20%

### 買入條件
- 週期溫度 < 30
- 且標的本身估值合理（不買垃圾股的反彈）
- 且有足夠的安全邊際

### 賣出條件
- 週期溫度 > 75（全面減倉）
- 或標的已達到合理估值（不再有安全邊際）
- 或發現自己的判斷基礎有誤

## 風險管理觀

1. **風險不可測量，只能判斷** — 不要迷信 VaR 等模型，用常識判斷
2. **最大的風險來自於大家認為沒有風險的時候** — 當 VIX 極低時要警覺
3. **不對稱性** — 追求「贏很多或虧很少」的機會，避免「贏很少但可能虧很多」
4. **保守是一種積極策略** — 在過熱的市場，不參與就是最好的操作

## 適用場景

- ✅ 最佳：市場極端時期（週期兩端）
- ✅ 適合：資產配置決策（倉位大小調整）
- ✅ 適合：加密貨幣（情緒波動大，週期明顯）
- ❌ 不適合：個股精選（Marks 更擅長宏觀判斷）
- ❌ 不適合：市場在中性區間時（沒有明確信號）
