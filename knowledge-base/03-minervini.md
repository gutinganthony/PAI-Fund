# Mark Minervini 馬克·乃爾維尼

> 「超級績效的秘密不是找到會漲的股票，而是在正確的時間點買入、在錯誤被證明時快速離場。」

## 核心理念

**一句話**：只在 Stage 2 上升趨勢中、且形成波動收斂（VCP）的標的上建倉，用嚴格的停損保護資本。

### 五大核心原則

1. **SEPA（Specific Entry Point Analysis）** — 在精確的進場點買入，不追高不猜底
2. **Stage Analysis** — 只買 Stage 2（上升趨勢），避開 Stage 1/3/4
3. **VCP（Volatility Contraction Pattern）** — 波動收斂後的突破是最佳買點
4. **風險優先** — 先想會虧多少，再想能賺多少。停損永遠在買入前就設好
5. **領導股原則** — 只交易所在板塊的領導者，不碰落後股

### 量化判斷

| Stage | 條件 | 操作 |
|-------|------|------|
| Stage 1（築底） | 價格在 200MA 附近震盪、成交量低 | 觀望 |
| Stage 2（上升）★ | 價格 > 150MA > 200MA、200MA 走平或上升 | 買入 |
| Stage 3（頂部） | 價格開始在 200MA 上方劇烈波動、量增價不漲 | 減倉/賣出 |
| Stage 4（下跌） | 價格 < 150MA < 200MA、200MA 下彎 | 空手/做空 |

## SEPA 趨勢模板（8 項必要條件）

全部滿足才考慮買入：

1. ✅ 當前價格 > 150 日均線（30 週均線）
2. ✅ 當前價格 > 200 日均線
3. ✅ 150 日均線 > 200 日均線
4. ✅ 200 日均線至少上升 1 個月（連續 22 個交易日走高）
5. ✅ 50 日均線 > 150 日均線 > 200 日均線
6. ✅ 當前價格 > 50 日均線
7. ✅ 當前價格距離 52 週低點至少 +30%
8. ✅ 當前價格距離 52 週高點不超過 -25%（越接近高點越好）

## VCP（波動收斂模式）

```
VCP 理想形態（3次收斂）：

價格
  │  ╱╲        第一次收斂 (幅度大，如 -20%)
  │ ╱  ╲   ╱╲        第二次收斂 (幅度縮小，如 -12%)
  │╱    ╲ ╱  ╲  ╱╲        第三次收斂 (幅度更小，如 -5%)
  │      ╲╱    ╲╱  ╲╱╲─── ★ 突破點（放量突破）
  │
  └────────────────────────── 時間
```

### 量化條件
- 至少 2 次收斂（收斂 = 回調幅度逐次縮小）
- 第一次收斂幅度 10-35%
- 最後一次收斂幅度 < 前一次的 50-60%
- 收斂期間成交量逐步萎縮（乾涸）
- 突破時成交量 > 50 日均量 × 1.5（放量確認）

## 可量化規則

### 買入條件（全部滿足）
- 通過 SEPA 8 項趨勢模板
- 出現 VCP 形態（至少 2 次收斂，最後收斂幅度 < 10%）
- 突破 VCP 最高點時放量（當日成交量 > 50 日均量 × 1.5）
- RS Rating > 80（相對強度排名前 20%）

### 停損規則（硬性）
- 買入後立即設定停損：買入價下方 7-8%
- 如果買入後 2-3 天沒有漲，但也沒觸發停損：縮緊停損至 5%
- 絕對不移動停損向下（只能上移）

### 賣出規則
- 觸發停損 → 無條件出場
- 盈利 20% → 賣出 1/3
- 價格跌破 50 日均線 → 賣出 1/2
- 價格跌破 150 日均線 → 全部出場
- Climax top（一日暴漲 15%+ 且成交量為近期最大）→ 賣出 1/2

## 加密貨幣適用調整

由於加密波動更大，參數需調整：
- 停損放寬至 10-12%（而非 7-8%）
- VCP 第一次收斂幅度可接受 20-50%
- 使用 4 小時線或日線（而非週線）
- RS Rating 改用同幣種板塊相對強度

## 風險管理觀

1. **單筆風險不超過總資金 1-2%** — 倉位大小由停損距離決定
2. **連續虧損 3 筆後暫停** — 重新審視市場環境
3. **最大持倉不超過 6-8 個標的** — 集中火力在最好的機會
4. **虧損是學費** — 小虧是正常的，大虧才是犯罪

## 適用場景

- ✅ 最佳：牛市初中期（大量個股進入 Stage 2）
- ✅ 適合：波段交易（持有數週到數月）
- ✅ 可適用：加密貨幣（調整參數後）
- ❌ 不適合：熊市（Stage 2 個股極少）
- ❌ 不適合：橫盤震盪市
