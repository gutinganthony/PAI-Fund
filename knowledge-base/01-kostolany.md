# André Kostolany 安德烈·科斯托蘭尼

> 「買股票，然後去吃安眠藥睡覺，等你醒來的時候，就會發現自己已經變得富有了。」

## 核心理念

**一句話**：市場由人性驅動，理解群眾心理週期就能在正確的時機站在正確的一邊。

### 五大核心原則

1. **雞蛋理論（Egg Theory）** — 市場在過熱與過冷之間循環，像雞蛋從一端滾到另一端
2. **固執投資人 vs 猶豫投資人** — 固執投資人有主見、有耐心、有資金；猶豫投資人跟風、恐慌、追漲殺跌
3. **貨幣 + 心理 = 趨勢** — 股市中期走勢由兩個因素決定：資金流動性和市場參與者的心理
4. **與群眾反向** — 當所有人都樂觀時要謹慎，當所有人都悲觀時要勇敢
5. **耐心是最大的美德** — 好的投資需要時間驗證，頻繁交易是敵人

## 雞蛋理論（量化框架）

```
                    過熱（賣出區）
                   /            \
              上漲中          開始修正
             /                    \
        底部復甦              恐慌拋售
             \                    /
              開始回暖      下跌加速
                   \            /
                    過冷（買入區）
```

### 階段判斷指標

| 階段 | 特徵 | 量化信號 |
|------|------|---------|
| 過冷/底部 | 成交量萎縮、媒體悲觀、VIX>30 | RSI<25, 價格<200MA*0.85, VIX>30 |
| 底部復甦 | 成交量微增、價格開始築底 | RSI 25-40, 價格開始>20MA |
| 上漲中 | 成交量增加、媒體轉正面 | RSI 40-65, 價格>50MA>200MA |
| 過熱 | 成交量暴增、全民炒股、媒體狂熱 | RSI>75, 價格>200MA*1.3, VIX<15 |
| 開始修正 | 量價背離、利多不漲 | RSI 背離, 成交量縮但價格創高 |
| 恐慌拋售 | 恐慌性拋售、投降式賣出 | VIX 急升>35, 連續跌停/大陰線 |

## 可量化規則

### 買入條件（任一觸發，需搭配確認）
- `VIX > 35` 且 `S&P500 RSI(14) < 30`（恐慌買入）
- 價格 < `200日均線 × 0.85` 且成交量 < `20日平均量 × 0.6`（過冷區間）
- 連續 5 日以上下跌後，出現第一根陽線且量能放大 > 前日 1.5 倍

### 賣出條件（任一觸發）
- `VIX < 15` 且 `RSI(14) > 75`（過熱賣出）
- 價格 > `200日均線 × 1.30` 且成交量連續 3 日萎縮
- 媒體情緒指標達到極端樂觀（Fear & Greed Index > 85）

### 不交易條件
- 市場在「上漲中」到「開始修正」的中間區段 — 等待方向明確
- 資金面（利率）與心理面（情緒）互相矛盾時 — 觀望

## 固執投資人 vs 猶豫投資人

| 特質 | 固執投資人（學習目標） | 猶豫投資人（避免） |
|------|----------------------|-------------------|
| 資訊 | 獨立思考，有自己的判斷 | 聽信小道消息 |
| 資金 | 用閒錢投資，不借錢 | 槓桿滿倉 |
| 耐心 | 可以等待數月甚至數年 | 天天看盤，頻繁操作 |
| 逆向 | 別人恐懼時貪婪 | 追漲殺跌 |
| 紀律 | 按計畫執行 | 情緒化決策 |

## 風險管理觀

1. **永遠不要全押** — 保留至少 30% 現金等待機會
2. **不要借錢投資** — 槓桿會把固執投資人變成猶豫投資人
3. **接受短期虧損** — 正確的判斷不一定馬上賺錢
4. **設定最大虧損容忍度** — 單筆不超過總資金 5%

## 適用場景

- ✅ 最佳：市場極度恐慌或極度貪婪時（雞蛋兩端）
- ✅ 適合：中長期投資（週線/月線級別）
- ❌ 不適合：盤整市場、短線交易
- ❌ 不適合：無明顯情緒極端的平淡行情

- 「只有少數人能在市場極度悲觀時買入，這就是為什麼只有少數人能賺大錢。」
