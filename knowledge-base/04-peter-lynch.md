# Peter Lynch 彼得·林區

> 「投資你了解的東西。」

## 核心理念

**一句話**：普通人可以透過日常觀察發現投資機會，關鍵是在合理的估值買入持續成長的企業。

### 五大核心原則

1. **生活選股法** — 從日常消費和觀察中發現好公司（你比華爾街更早知道哪家店生意好）
2. **PEG 估值法** — P/E 除以盈餘成長率，PEG < 1 表示成長被低估
3. **六種股票分類** — 不同類型的股票用不同策略
4. **研究是關鍵** — 買股票前至少要能用 2 分鐘說清楚為什麼買
5. **不要預測市場** — 花在預測市場走向的時間不如花在研究個股

## 六種股票分類與策略

| 類型 | 特徵 | 策略 | 預期回報 |
|------|------|------|---------|
| 緩慢成長股 | 營收成長 < 5%，高股息 | 股息收入，價值低估時買入 | 年化 6-8% |
| 穩定成長股 | 營收成長 5-12%，大型企業 | 在低估時買入，漲 30-50% 賣出 | 年化 10-15% |
| 快速成長股 ★ | 營收成長 > 20%，中小型 | 核心持倉，持有到成長放緩 | 年化 20-50% |
| 景氣循環股 | 跟經濟週期高度相關 | 在週期低點買，高點賣 | 波段 30-100% |
| 轉機股 | 財務困難但有改善跡象 | 高風險高回報，需深入研究 | 不確定 |
| 資產股 | 帳面資產被市場低估 | 資產被發現時會重估 | 不確定 |

## 可量化規則

### PEG 篩選（核心指標）

```
PEG = P/E ÷ 盈餘成長率(%)

PEG < 0.5  → 嚴重低估 ★★★ 強烈買入
PEG 0.5-1.0 → 低估 ★★ 考慮買入
PEG 1.0-1.5 → 合理估值 — 持有
PEG 1.5-2.0 → 偏貴 — 考慮減倉
PEG > 2.0   → 高估 — 不買 / 賣出
```

### 快速成長股篩選條件（美股）
- 營收成長率 > 20%（近 3 年 CAGR）
- 盈餘成長率 > 20%（近 3 年 CAGR）
- PEG < 1.0
- 負債/權益比 < 0.5
- 市值 < 500 億美元（太大的公司難以快速成長）
- 機構持股比 < 60%（機構尚未完全發現）

### 穩定成長股篩選條件（美股）
- 營收成長率 5-15%（穩定）
- P/E < 行業平均
- 股息率 > 1.5%
- 自由現金流持續為正
- 大型企業（市值 > 100 億美元）

### 買入條件
- 通過分類對應的篩選條件
- 能用「2 分鐘故事」說清楚為什麼買（有清楚的投資邏輯）
- PEG < 1.0 或接近歷史低位

### 賣出條件
- 快速成長股：營收成長率連續 2 季 < 15%（成長放緩）
- 穩定成長股：漲幅達 30-50% 且 PEG > 1.5
- 所有類型：原始買入理由不再成立
- 找到更好的機會（機會成本）

## 「2 分鐘故事」模板

每筆交易都要能回答：
1. 我為什麼買這家公司？（一句話）
2. 它屬於六種分類中的哪一種？
3. 它的成長引擎是什麼？
4. PEG 是多少？相對歷史水平如何？
5. 最大的風險是什麼？

## 風險管理觀

1. **分散但不過度** — 持有 6-12 檔，太多等於買指數
2. **不要買你不懂的** — 如果不能用簡單語言解釋，就不要買
3. **接受波動** — 股價下跌 25% 很正常，不代表你判斷錯誤
4. **定期檢視故事** — 每季重新講一次「2 分鐘故事」，講不出來就賣

## 適用場景

- ✅ 最佳：發現尚未被華爾街注意的成長股
- ✅ 適合：中長期投資（持有數月到數年）
- ❌ 不適合：加密貨幣（沒有營收/盈餘可分析）
- ❌ 不適合：短線交易
